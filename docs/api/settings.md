## Update Settings

- Address PUT `/api/settings`
- Request Type application/json
- Params
  - token
  - telegramName
  - website
  - bio
  - emailNotification: Whether to receive email notifications
- Return Result()

```json
{
  "code": 200,
  "description": "SUCCESS",
  "detail": null
}
```

- Params

- Return Result(String) // Return new token
